# Краткое руководство: Функция "Рабочие места"

## 🎯 Что это такое?

Функция позволяет указать, к какому конкретному компьютеру в кабинете относится проблема. Это очень полезно, когда в одном помещении несколько рабочих мест.

## 👤 Для обычных пользователей

### Как создать тикет с указанием рабочего места?

1. **Нажмите "Создать тикет"**
2. **Заполните форму**:
   - Заголовок
   - Описание проблемы
   - Приоритет
   - **Рабочее место/Компьютер** ← Выберите здесь нужное рабочее место
   - Срок выполнения (опционально)

3. **Нажмите "Создать"**

### Что отображается для каждого рабочего места?

```
Кабинет 101 (IT кабинет)
├─ ПК-1 (Левый угол, у окна)
├─ ПК-2 (Центр, основной стол)
└─ ПК-3 (Правый угол)

Кабинет 102 (Офис)
├─ ПК-1 (Левый край)
└─ ПК-2 (Центр)

Кабинет 103 (Лаборатория)
├─ Рабочее место 1 (Стол 1)
└─ Рабочее место 2 (Стол 2)
```

## 🛠️ Для администраторов

### Как добавить новое рабочее место?

1. **Перейдите в админ-панель**: http://localhost:8000/admin/
2. **Найдите "Рабочее место"** в левом меню
3. **Нажмите "Добавить рабочее место"**
4. **Заполните данные**:
   - **Кабинет/Офис**: Номер или название помещения (например: "101", "102 (Офис)", "Кабинет учителя")
   - **Номер/Описание**: Номер компьютера (например: "ПК-1", "Рабочее место 1", "Левый стол")
   - **Местоположение**: Дополнительное описание (например: "У окна", "Левый угол")

5. **Сохраните**

### Как редактировать рабочее место?

1. **Перейдите в админ-панель**
2. **Найдите "Рабочее место"**
3. **Кликните на нужное рабочее место**
4. **Отредактируйте данные**
5. **Сохраните**

### Как фильтровать рабочие места?

На странице управления рабочими местами есть фильтр по кабинетам. Выберите кабинет в правой части экрана, чтобы увидеть только рабочие места этого кабинета.

### Как искать рабочее место?

Используйте строку поиска вверху для поиска по номеру или описанию рабочего места.

## ❓ Часто задаваемые вопросы

**Q: Обязательно ли указывать рабочее место?**
A: Нет, это опциональное поле. Вы можете оставить его пустым.

**Q: Что произойдет, если рабочее место будет удалено?**
A: Поле рабочего места в тикете просто станет пустым. Сам тикет не будет удален.

**Q: Могу ли я создать два рабочих места с одинаковым номером в одном кабинете?**
A: Нет, система это не позволит. Каждое рабочее место уникально в рамках кабинета.

**Q: Как быстро найти все проблемы на конкретном рабочем месте?**
A: В админ-панели на странице "Рабочее место" нажмите на нужное рабочее место, и вы увидите все связанные тикеты.
