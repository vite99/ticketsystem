## Добавление своих рабочих мест

### Способ 1: Через админ-панель (Рекомендуется)

1. Откройте http://localhost:8000/admin/
2. Нажмите на "Рабочее место"
3. Нажмите "Добавить рабочее место"
4. Заполните:
   - **Кабинет/Офис**: Номер или название (например: "205", "Офис A")
   - **Номер/Описание**: Номер компьютера (например: "ПК-5", "Рабочее место")
   - **Местоположение**: Описание (например: "У окна", "Левый стол")
5. Нажмите "Сохранить"

### Способ 2: Через Python скрипт

Отредактируйте `create_workstations.py`:

```python
workstations_data = [
    {'room': 'Ваш кабинет 1', 'number': 'ПК-1', 'location': 'Ваше описание'},
    {'room': 'Ваш кабинет 1', 'number': 'ПК-2', 'location': 'Ваше описание'},
    # ... добавьте ваши
]
```

Затем запустите:
```bash
.venv\Scripts\python create_workstations.py
```

## Устранение проблем

### ❌ Проблема: "No such table: tickets_workstation"

**Решение**:
```bash
.venv\Scripts\python manage.py migrate
```

---

### ❌ Проблема: Поле workstation не видно в форме создания

**Решение**: Убедитесь, что файл `forms.py` содержит:
```python
fields = ['title', 'description', 'priority', 'workstation', 'due_date']
                                          ↑
                                      Это должно быть там
```

---

### ❌ Проблема: Ошибка в админ-панели

**Решение**: Проверьте, что в `admin.py` есть:
```python
admin.site.register(Workstation, WorkstationAdmin)
```

---

### ❌ Проблема: Рабочие места не создаются

**Решение**:
```bash
# Проверьте, что БД пустая или существует
.venv\Scripts\python manage.py dbshell

# В консоли SQLite
sqlite> SELECT COUNT(*) FROM tickets_workstation;

# Если 0, запустите:
.venv\Scripts\python create_workstations.py
```
