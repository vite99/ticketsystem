# 👤 Страница профиля пользователя для администраторов

## Описание

Новая функция позволяет администраторам просматривать полный профиль пользователя и всю историю его тикетов в одном месте.

## Как это работает

### 1. **Основная информация**
На странице профиля отображается:
- ✅ Полное имя и email пользователя
- ✅ Отдел и кабинет (из UserProfile)
- ✅ Телефон контакта
- ✅ Рабочее место (компьютер)
- ✅ Статус одобрения

### 2. **Статистика**
Три карточки со статистикой:
- 📊 **Всего тикетов** - общее количество созданных и назначенных тикетов
- 🟠 **Открыто** - количество новых, в работе и ожидающих тикетов
- 🟢 **Закрыто** - количество решённых и закрытых тикетов

### 3. **Таблица тикетов**
Полная история всех тикетов пользователя:
- ID тикета (кликабельная ссылка)
- Заголовок (кликабельная ссылка)
- Статус (цветной бейдж)
- Приоритет (цветной бейдж)
- Дата создания

Таблица отсортирована по дате создания (новые сверху).

## Как добавить ссылку на профиль

### Способ 1: Из списка тикетов (ticket_list.html)
```html
<a href="{% url 'user_detail_admin' ticket.creator.id %}">
    {{ ticket.creator.username }}
</a>
```

### Способ 2: Из деталей тикета (ticket_detail.html) - УЖЕ ДОБАВЛЕНО ✅
Ссылка на имя создателя и назначенного пользователя уже активна для администраторов.

### Способ 3: Из админ-панели
Любой другой части приложения можно добавить:
```html
<a href="{% url 'user_detail_admin' user_id %}">
    Посмотреть профиль
</a>
```

## Навигация в приложении

**URL**: `/users/<user_id>/`

**Примеры**:
- `/users/1/` - профиль пользователя с id=1
- `/users/42/` - профиль пользователя с id=42

**Возвращение назад**: 
- Кнопка "← Назад к списку тикетов" вернет на главную страницу

## Требования доступа

✅ **ТОЛЬКО для администраторов** (`is_staff=True`)

Обычные пользователи:
- ❌ Не могут открыть страницу профиля другого пользователя
- ❌ Перенаправляются на страницу входа или на главную

## Интеграция в другие места

### 1. Список тикетов (ticket_list.html)

Добавьте ссылку на имя создателя:

```html
<!-- БЫЛО: -->
<td>
    <small>{{ ticket.creator.username }}</small>
</td>

<!-- СТАЛО: -->
<td>
    <small>
        {% if user.is_staff %}
            <a href="{% url 'user_detail_admin' ticket.creator.id %}">
                {{ ticket.creator.username }}
            </a>
        {% else %}
            {{ ticket.creator.username }}
        {% endif %}
    </small>
</td>
```

### 2. Админ-панель одобрения пользователей

```html
<a href="{% url 'user_detail_admin' user.id %}" class="btn btn-info">
    👁️ Посмотреть профиль
</a>
```

### 3. Любая другая ссылка

```html
<a href="{% url 'user_detail_admin' user.id %}">
    Профиль: {{ user.get_full_name }}
</a>
```

## Часто задаваемые вопросы

**Q: Могут ли обычные пользователи видеть профили друг друга?**
A: Нет, только администраторы с `is_staff=True`.

**Q: Где найти прямую ссылку на профиль пользователя?**
A: В детале тикета нажмите на имя создателя или назначенного (если вы администратор).

**Q: Сколько тикетов отображается на одной странице?**
A: 20 тикетов (пагинация).

**Q: Отображаются ли все тикеты пользователя?**
A: Да, все созданные И назначенные тикеты в одном списке.

**Q: Как отфильтровать только открытые тикеты?**
A: На момент это не реализовано, можно добавить фильтр в будущих версиях.
