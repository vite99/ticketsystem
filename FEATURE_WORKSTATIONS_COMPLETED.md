## 🎯 Функциональность

### Для пользователей
- При создании тикета выбирают рабочее место из выпадающего списка
- На странице деталей тикета видят информацию о рабочем месте
- В списке тикетов (для администраторов) видно рабочее место проблемы

### Для администраторов
- Полное управление рабочими местами через админ-панель
- Возможность добавления новых кабинетов и рабочих мест
- Фильтрация и поиск по рабочим местам
- Просмотр всех тикетов, привязанных к конкретному рабочему месту

## 🔒 Целостность данных

- Уникальность: Невозможно создать два рабочих места с одинаковым именем в одном кабинете
- Защита: При удалении рабочего места, поле workstation в тикетах становится пустым (не удаляется тикет)
- Валидация: Поле опциональное (blank=True), что позволяет старым тикетам остаться без рабочего места
