## 🚀 Как начать использовать

### Для администраторов:
1. Откройте админ-панель: http://localhost:8000/admin/
2. Перейдите в раздел "Рабочее место"
3. Просмотрите созданные примеры или добавьте свои

### Для пользователей:
1. При создании тикета выберите нужное рабочее место из выпадающего списка
2. Сохраните тикет
3. Администратор увидит точное местоположение проблемы
